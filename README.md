## 🌟 特点

这是一个为 RPG Maker MV / MZ 游戏开发的资源还原解密工具。能够帮助你一键解密并还原游戏中的加密图片（PNG）和音频（OGG、M4A）资源。

*   **绿色单文件**：软件为单文件版本，双击即可直接运行，无需安装多余环境。
*   **自动提取密钥**：自动扫描游戏配置文件，识别并填入解密密钥。
*   **解密后即开即玩**：解密时会自动修补导出目录下的游戏配置文件。解密完成后，新生成的游戏文件夹可以直接运行游玩，无需手动修改配置。

---

## 📂 使用方法

### 第 1 步：启动工具
双击运行 **`rpg-decrypter.exe`**。
启动后会弹出一个命令行窗口，并在浏览器中打开操作界面。
> *如果浏览器没有自动弹出，可以手动在浏览器地址栏输入命令行窗口中显示的网址（通常为 `http://localhost:3000`）。*

### 第 2 步：选择目录与扫描
1. 复制你想要解密的游戏文件夹根目录（即包含 `Game.exe` 的文件夹）的绝对路径。
2. 将路径粘贴到工具界面的 **“游戏目录”** 输入框中，点击 **“扫描目录”**。
3. 扫描完成后，工具会自动读取并填入密钥。

### 第 3 步：一键还原解密
1. 点击 **“开始还原解密”** 按钮。
2. 进度条会实时更新，显示当前的文件解密进度。

### 第 4 步：查看解密成果
解密完成后，点击 **“打开解密目录”** 按钮，可以直接弹出解密后的文件夹。
> *解密后的文件夹默认会创建在原游戏文件夹的同级目录下，并命名为 `[原游戏文件夹名]_decrypted`。*
