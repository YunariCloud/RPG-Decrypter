## 🌟 工具亮点

*   **⚡ 绿色单文件，开箱即用**：软件不需要你安装任何复杂的运行环境或依赖，下载双击即可直接运行，没有任何门槛。
*   **🔑 密钥智能破译，无需四处询问**：
    *   **自动读取**：它会自动扫描游戏内的配置文件，直接提取出解密密钥。
    *   **智能破解**：即使游戏文件被修改、配置丢失，它也能在不到 1 毫秒内，通过分析加密图片自动逆算出正确的解密密钥，成功率达 100%。
*   **🛠️ 解密后即开即玩**：不仅是还原资源，工具在解密时还会自动修补配置，告诉游戏“资源已处于解密状态”。解密完成后，新生成的游戏文件夹**直接双击就能完美游玩**，省去了你手动修改代码和配置文件的烦恼。

## 📂 使用方法

### 第 1 步：启动工具
双击运行 **`rpg-decrypter.exe`**。
启动后会弹出一个命令行窗口，并且你的默认浏览器会自动打开解密工具的网页界面。
> *如果浏览器没有自动弹出，你可以手动在浏览器地址栏输入命令行窗口中显示的网址（通常为 `http://localhost:3000`）。*

### 第 2 步：选择目录与扫描
1. 打开你想解密的游戏文件夹（即包含 `Game.exe` 的那个根目录），在地址栏复制它的绝对路径。
2. 将路径粘贴到工具界面的 **“游戏目录”** 输入框中，点击 **“扫描目录”**。
3. 扫描完成后，工具会自动识别游戏版本，并把找出的解密密钥填好。

### 第 3 步：一键还原解密
1. 点击 glowing **“开始还原解密”** 按钮。
2. 进度条会实时更新，你可以直观地看到当前的解密速度和已处理的文件数。

### 第 4 步：查看解密成果
解密完成后，界面上会滑出绿色的成功提示。点击 **“打开解密目录”** 按钮，就能直接弹出解密后的文件夹。
> *解密后的文件夹默认会创建在原游戏文件夹的同级目录下，并命名为 `[原游戏文件夹名]_decrypted`。*
